2.В первом ящике находится 8 мячей, из которых 5 - белые. Во втором ящике - 12 мячей, из которых 5 белых. Из первого ящика вытаскивают случайным образом два мяча, из второго - 4. Какова вероятность того, что 3 мяча белые?

1) Два белых из первого ящика

(5/8 * 4/7) * (5/12 * 7/11 * 6/10 * 5/9)*4 = 0.13

2) Один белый из первого ящика

(5/8 * 3/7)*2 * (5/12 * 4/11 * 7/10 * 6/9)*12 = 0.45

3) Два чёрных из первого ящика

(3/8 * 2/7) * (5/12 * 4/11 * 3/10 * 7/9)*4 = 0.015

0.13 + 0.45 + 0.015 = 0.6
